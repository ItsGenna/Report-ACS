# Advanced Control Systems Report

This repository contains the final project and report for the Advanced Control Systems course at RWTH Aachen University. My teammate and I developed this project over the course of the semester, achieving the highest grade in the class.

## Repository Structure

* **`Questions.pdf`**: The assignment questions and problem statements.
* **`Final report.pdf`**: The complete report detailing our methodology, control strategies, and final results.
* **`matlab/`**: The MATLAB scripts and code used for our calculations and simulations.
* **`report/`**: The LaTeX source code used to generate the final PDF report.
